quippy.util.quip_xml_parameters(name, label=None)[source]¶

Search for an QUIP XML parameter set matching name’ and, optionally `label.

Result is the XML stanza as a string. Looks under ${QUIP_ROOT}/share/Parameters for a file matching name, name.xml, ip.parms.`name`.xml or tightbind.params.`name`.xml. If label is given, it should be a text string matching the label attribute of a QUIP <params> XML stanza. If a matching parameter set can’t be found, we raise an IOError.

quippy.util.analyse_timings(lines)[source]¶

Analyse QUIP timing information.

Lines should be list of lines read from output file, with e.g. readlines(). Returns two dictionaries (timings, summary). timings contains entries for each TIMING line, with “wall” and “cpu” keys. summary collapses repeated entries and adds statistics for th enumber of calls (“len” key), total time (“sum”), and average time per call (“mean”).
